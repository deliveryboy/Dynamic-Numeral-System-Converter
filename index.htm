Dynamic Numeral System Converter

Converts from and to any of the given numeral systems below (integer only).
The supported numeral systems are Binary, Octal, Decimal and Hexadecimal respectively.

Binary [0-1]
Octal [0-7]
Decimal [0-9]
Hexadecimal [0-9,A-F]

Known bugs

  • It is still possible to type in some invalid characters.
  • If a large decimal integer will be displayed in exponential notation the non fractional part omitting the exponent will be evaluated only.
  • The use of numpad is not supported.
